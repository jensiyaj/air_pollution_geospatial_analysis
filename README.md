# Geospatial Analysis of Air Pollution

## Goal
To analyze and visualize air quality data on interactive maps to identify pollution hotspots and understand geospatial trends across different seasons.

## Tools Used
Python, Pandas, GeoPandas, Folium, Matplotlib

## Key Insights
1. Processed city-wide air quality datasets to map PM2.5 concentration levels
2. Identified pollution hotspots using geospatial visualization on interactive Folium maps  
3. Analyzed correlation between industrial zones and high AQI levels across seasons
4. Derived location-based insights for pollution pattern analysis

## Dataset
Used city air quality monitoring data with latitude, longitude, and PM2.5 values.
